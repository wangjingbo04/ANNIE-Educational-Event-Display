# ANNIE Educational Event Display

Interactive educational simulator for visualizing neutrino interactions in the ANNIE detector.

---
**Disclaimer**

This project is an educational simulator developed for outreach and teaching.

It is **not** the official ANNIE event display or reconstruction software.

Detector geometry, detector response, and event generation are intentionally simplified for visualization and educational purposes.
---

## Description

This project is an educational and outreach event display designed to demonstrate how neutrino interactions are reconstructed in the ANNIE detector.

## Current Features

- Interactive 3D detector visualization
- Cylindrical water Cherenkov detector
- >100 PMTs
- Front Muon Veto (FMV)
- Muon Range Detector (MRD)
- Muon track visualization
- Cherenkov light visualization
- PMT charge simulation
- Scientific-style 2D event display
- Export event display to PDF
- Camera presets and presentation tools

## Future Development

## Running Locally

Open `index.html` in a modern web browser. The simulator uses HTML, CSS, vanilla JavaScript, and Three.js with no backend.
